Visión Artificial

Detección y lectura automática de placas vehiculares mediante cámara y procesamiento de imagen en tiempo real.

Control de Barrera

Apertura automática de la barrera cuando el sistema detecta una placa autorizada en la base de datos.

Notificación WhatsApp

Alertas automáticas vía chatbot al número registrado cada vez que el vehículo ingresa o sale del campus.

Control de Seguridad

Registro y gestión de vehículos no autorizados con intervención del personal de seguridad del campus.

Descripción del Sistema

Este sistema permite detectar automáticamente la placa de un vehículo usando visión artificial. Fue desarrollado para la Universidad Libre Seccional Cúcuta.

Funcionalidad principal: Al detectar una placa registrada, el sistema abre la barrera automáticamente y envía una notificación por WhatsApp al propietario. Versión 2 del proyecto.
