🎥 Visión Artificial

Detección automática de placas vehiculares mediante algoritmos de procesamiento de imagen en tiempo real. (RU-SI-01, 02, 03)

💬 Notificación WhatsApp

Alertas automáticas vía chatbot de WhatsApp sobre entradas y salidas de vehículos registrados. (RU-SI-10)

🚧 Control de Barrera

Apertura automática de barreras para vehículos autorizados registrados en la base de datos. (RU-SI-04, 06)

🛡️ Seguridad

Registro y gestión de vehículos no autorizados con notificación al personal de seguridad. (RU-ED-01, 02)

Descripción del Sistema

Este sistema permite detectar automáticamente la placa de un vehículo usando visión artificial. Fue desarrollado para la Universidad Libre Seccional Cúcuta.

Funcionalidad principal: Al detectar una placa registrada, el sistema abre la barrera automáticamente y envía una notificación por WhatsApp al propietario. La versión actual es la 2ª entrega del proyecto.

Características del Sistema

Características principales

  • Detección de placas en tiempo real (RU-SI-01, RU-SI-02)
  • Análisis y reconocimiento de texto OCR en placas (RU-SI-03)
  • Verificación de autorización en base de datos (RU-SI-04)
  • Apertura automática de barrera para autorizados (RU-SI-06)
  • Registro de todos los eventos de acceso con foto, fecha y hora (RU-SI-05)
  • Detección y registro de vehículos no autorizados (RU-SI-07, RU-SI-08)
  • Registro de placas no encontradas en la base de datos (RU-SI-11)
  • Notificación vía WhatsApp al usuario registrado (RU-SI-10)
  • Carga masiva de usuarios por CSV o XLS (RU-SI-13, RU-AD-03)
  • Registro individual de usuarios (RU-SI-14)
  • Interfaz web de administración (RU-AD-01, RU-AD-02)

Pasos de funcionamiento

  1. La cámara captura la imagen del vehículo en la entrada o salida (RU-SI-01)
  2. El algoritmo de visión artificial detecta y lee la placa (RU-SI-02, RU-SI-03)
  3. El sistema consulta la base de datos de vehículos registrados (RU-SI-04)
  4. Si la placa es autorizada, se abre la barrera automáticamente (RU-SI-06)
  5. Si la placa no está registrada o no tiene permisos, se alerta al personal de seguridad (RU-SI-07, RU-ED-01)
  6. Se registra el evento con placa, foto, fecha, hora y tipo de movimiento (RU-SI-05, RU-SI-08)
  7. Se envía notificación por WhatsApp al propietario del vehículo (RU-SI-10)

Glosario del sistema

Placa autorizada
Vehículo registrado con permisos vigentes en la base de datos.
Placa no autorizada
Vehículo que no está registrado o cuyo permiso ha expirado.
Evento de acceso
Registro de cada entrada o salida con fecha, hora, imagen y estado.
Chatbot WhatsApp
Bot que envía notificaciones automáticas al usuario cuando su vehículo ingresa o sale.

Términos y condiciones (RU-US-02)

Al registrarse, acepta que:

  • Sus datos personales y de vehículo serán almacenados en la base de datos de la Universidad Libre Seccional Cúcuta.
  • Se capturarán imágenes fotográficas de su vehículo cada vez que ingrese o salga del campus.
  • Recibirá notificaciones automáticas vía WhatsApp sobre los eventos de acceso de su vehículo.
  • El acceso al parqueadero podrá ser revocado por la administración en cualquier momento.
  • La información solo será usada para fines de seguridad y control de acceso.

Panel de Administrador

Módulo de gestión exclusivo para el administrador del sistema. Permite habilitar, inhabilitar y administrar vehículos y usuarios. (RU-AD-01, RU-AD-02, RU-AD-03, RU-AD-21)

➕ Habilitar Vehículo (RU-AD-01)

Agregar un nuevo vehículo autorizado a la base de datos.

🚫 Inhabilitar Vehículo (RU-AD-02)

Revocar el acceso de un vehículo registrado en el sistema.

📂 Carga Masiva de Usuarios (RU-AD-03 / RU-SI-13)

Cargar un archivo en formato CSV o XLS con los datos de los usuarios. El sistema validará la estructura, el formato y los campos obligatorios antes de procesar el archivo.

📋 Campos requeridos en el archivo: Cédula, Nombre completo, Tipo de usuario, Programa/Dependencia, Placa, Fecha inicio, Fecha fin.

👤 Registro Individual (RU-SI-14)

Registrar un único usuario directamente desde el panel sin necesidad de archivo.
