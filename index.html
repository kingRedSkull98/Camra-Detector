Visión Artificial

Detección automática de placas vehiculares mediante algoritmos de procesamiento de imagen en tiempo real.

Notificación WhatsApp

Alertas automáticas vía chatbot de WhatsApp sobre entradas y salidas de vehículos registrados.

Control de Barrera

Apertura automática de barreras para vehículos autorizados registrados en la base de datos.

Descripción

Este sistema permite detectar automáticamente la placa de un vehículo usando visión artificial. Fue desarrollado para la Universidad Libre Seccional Cúcuta.

Funcionalidad principal: Al detectar una placa registrada, el sistema abre la barrera automáticamente y envía una notificación. La versión actual es la 1ª entrega del proyecto.

Características del sistema

Características principales

  • Detección de placas en tiempo real
  • Base de datos de vehículos autorizados
  • Apertura automática de barrera
  • Notificación vía WhatsApp
  • Interfaz web de administración

Pasos de funcionamiento >

  1. La cámara captura la imagen del vehículo en la entrada
  2. El algoritmo de visión artificial detecta y lee la placa
  3. El sistema consulta la base de datos de vehículos registrados
  4. Si la placa es autorizada, se abre la barrera automáticamente
  5. Se registra la entrada/salida y se envía notificación por WhatsApp
